I want to be a Full Stack Web Developer

What does "Full" in Full Stack mean?

Full means that you develop both the front end and the back end. The front end is what the users sees on what is called the client computer. The back end is what is on the server computer that delivers content to the front end

What does "Stack" in Full Stack mean?

Stack means that several programs, a "stack" of programs is used. On the front end I will use HTML5, CSS3, and JavaScript. On the back end I use PHP, and MySQL. And I also have a LAMP stack on my server computer.

What does "LAMP Stack" mean

LAMP is a stack used for our web page server - remember, a server sends web pages to a client. LAMP Stands for: L = Linux, the basis of our server operating system A = Apache, our web server M = MySQL, the database we use, P = PHP, the programming language used on the server side.
